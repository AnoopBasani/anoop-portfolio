Supply Chain Delay Tracker

Tracking vendor delays and SLA performance

Overview

Dashboard built to help operations monitor late deliveries, vendor performance, and automate alerts when thresholds are breached.

Tools Used

  • Power BI
  • SQL Server
  • Power Automate

Highlights

  • Created automated alerts for delay thresholds
  • Mapped supply chain KPIs with Power BI
  • Integrated SLA tracking reports

Sample SQL

SELECT shipment_id, vendor_id, DATEDIFF(day, dispatch, delivery) AS delay FROM logistics.shipments;
