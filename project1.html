Hourly Sales Projection Dashboard

Real-time sales insights for fast-paced decision making

Overview

This dashboard allows leadership to monitor hourly sales and compare them to previous benchmarks, enabling rapid decision making and campaign adjustments.

Tools Used

  • Power BI
  • BigQuery
  • Power Automate
  • Control-M

Highlights

  • Built dynamic DAX for YoY comparisons
  • Used Power Automate for alerting
  • Scheduled refresh with Control-M

Sample SQL

SELECT store_id, DATE(sale_time) AS sale_date, EXTRACT(HOUR FROM sale_time) AS hour, SUM(sale_amount) AS total FROM `dataset.sales` GROUP BY store_id, sale_date, hour;
