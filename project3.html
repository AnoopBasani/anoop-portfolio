Customer Behavior Analysis

Unlocking behavioral insights with RFM & cohort analysis

Overview

This Tableau dashboard visualizes RFM-based segmentation and behavioral trends to assist marketing teams in targeting high-value customer groups.

Tools Used

  • Tableau
  • SQL
  • BigQuery

Highlights

  • Built RFM models with SQL
  • Interactive cohort heatmaps
  • Churn analysis & funnel breakdowns

Sample SQL

SELECT customer_id, MAX(purchase_date) AS last, COUNT(*) AS frequency, SUM(amount) AS value FROM transactions GROUP BY customer_id;
